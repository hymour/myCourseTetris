Kortris

Добро пожаловать на мою курсовую работу по дисциплине Программирование (Дополнительные главы)

Это, скажем так, "мой" с нуля написанный тетрис на open-source движке Godot

Чтобы просто начать играть, скачайте релиз в Releases и распакуйте архив, а после запустите игру и нажмите Play

Highscore сохраняется в стандартное место для Godot: Users\<имя пользователя>\AppData\Roaming\Godot\app_userdata\myCourseTetris\highscore.save

Управление:

Стрелочка Вверх - Hard-drop фигуры вниз

Стрелочка Вправо и Влево - перемещение фигуры влево и вправо

Стрелочка Вниз (Зажать) - быстрое перемещение вниз

Z - кнопка поворота влево

X - кнопка поворота вправо

Esc - пауза
